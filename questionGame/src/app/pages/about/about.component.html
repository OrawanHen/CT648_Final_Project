How to start

Go to Open Trivia Database (opentdb.com) for get API
1. Enter the desired value.
2. Click "Generate API URL".
3. Copy the API URL shown under "API URL Generated!".
4. Paste the copied API URL into the "input API" field on the New Game Page.
5. Click "Check"
6. The question game will appear, and you can enjoy playing!

Thing you need to know After check

1. Correct Answer: This means your answer is correct.
2. Incorrect Answer: This means your answer is wrong.
3. Try Again: This means you need to try answering the question again.

What I Do

My platform allows users to enjoy creating and participating in multiple-choice quizzes. By connecting to the Open Trivia Database API, I bring you an array of carefully curated questions across different subjects. Whether you're playing for fun, testing your knowledge, or challenging friends, my quiz game is designed to be both entertaining and insightful.

Key Features

Play All Games: Users have the option to browse and participate in a variety of quizzes. These quizzes cover multiple categories and difficulty levels, providing an engaging challenge for all players.
Create New Games:Users can start new games and customize their quiz experience by selecting categories, number of questions, and difficulty levels. This feature ensures a personalized and unique game each time.
History and Points Tracking: Players can view their game history, including scores and points earned. This feature helps users keep track of their progress and motivates them to improve with each game.

Technology

I developed this project using the Angular framework, known for its robust features and efficient performance in building modern web applications. By leveraging Angular's capabilities, I ensure a seamless and responsive user experience.

For backend operations, I use Node.js to handle server-side logic and API management. To connect and manage user data and quiz questions, I rely on PostgreSQL, a powerful and reliable open-source relational database system. This ensures secure and efficient storage and retrieval of user information and game data.

Deployment

For deployment, I use Docker Compose to streamline the process and create a reliable, multi-container environment. My setup includes:
Frontend (angular-app) : An Angular app that serves as the user-facing part of the quiz game.
Backend Server (server) : A server application responsible for handling user interactions and connecting with the database.
Database (db) : A data storage system for user login information and quiz questions, ensuring smooth and secure access.
